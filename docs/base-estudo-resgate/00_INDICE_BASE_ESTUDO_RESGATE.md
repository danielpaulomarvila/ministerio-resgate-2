# Base de Estudo e Conferência — Ministério Resgate / Família Resgate 2.0

> Documento-guia para usar os projetos antigos como histórico, pesquisa e conferência antes de novas etapas.

## Origem da análise

Repositórios de referência:

1. `danielpaulomarvila/rederesgate-cancelado-pesquisa`
2. `danielpaulomarvila/sistema-igreja`

## Objetivo

Guardar, organizar e melhorar tudo que pode ser reaproveitado no projeto atual, sem copiar erros antigos.

Esta base serve para:

- conferir se alguma ideia importante ficou esquecida;
- evitar repetir erros de arquitetura, mock/localStorage, duplicidade e páginas soltas;
- orientar próximas etapas do Família Resgate;
- preparar backend real depois da aprovação visual;
- separar corretamente membro comum, jovem/resgatado, responsável, liderança, secretaria e administração;
- consolidar regras de pontuação, destaques, intercessão, financeiro e perfil.

## Regra de ouro

Tela aprovada visualmente pode usar mock temporário.

Tela entregue para uso real não pode depender de mock, localStorage falso ou dados decorativos.

Mock é andaime. Backend real é estrutura.
